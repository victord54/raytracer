# Ray tracer
This is a simple ray tracer written in C++.

## Features
It uses the following concepts :
- Ray tracing
- Phong shading
- Reflection
- Refraction
- Anti-aliasing
- Soft shadows
- Depth of field

## Build
```bash
mkdir build
cd build
cmake ..
make
```

## Run
```bash
./raytracer
```

## Output
The output is a .ppm file that can be visualized with an image viewer.
